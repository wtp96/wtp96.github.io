當你開始閱讀一段文字時，第一眼注意到的往往不是內容，而是整個頁面散發出來的氛圍。留白、字距、行高，還有背景顏色，這些細節會在潛意識裡告訴大腦：這裡適合慢慢讀，還是適合快速掃過。

## 為什麼排版會影響專注力

人類的眼睛並不喜歡在一行文字上跑太遠。當行長超過某個範圍，讀者就會在換行時迷失位置，閱讀節奏被打斷。把文字控制在舒適的寬度內，就像幫讀者劃出一條專注的小徑，讓視線可以自然從一行滑到下一行。

除了寬度，垂直方向的節奏也同樣重要。段落之間保留足夠的呼吸空間，標題與內文之間有清楚的層次，這些都會讓人願意繼續往下滾動。

> 好的閱讀體驗不是把資訊塞滿畫面，而是讓讀者在每一刻都覺得輕鬆。

## 一些常見的排版元素

在實際寫作時，我們會用到各種元素來輔助表達：

- **清單**可以拆分行列，降低一次要處理的資訊量
- **引用**能讓重要觀點跳出來，製造語氣上的變化
- **程式碼**則是技術部落格不可或缺的元素

下面是一段簡單的程式碼範例：

```js
function focusOnReading() {
  const atmosphere = 'calm';
  const typography = 'generous';
  return atmosphere && typography;
}
```

## 讓人想繼續讀下去

沉浸感來自於一致性。背景、文字、連結、程式碼區塊，所有元素的色調都應該和諧。當畫面沒有突兀的對比，讀者就不會被視覺干擾拉走注意力。

最後，適當的字型大小與行高能讓長篇內容變得不那麼沉重。你可以試著把這篇文章從頭讀到尾，感受一下眼睛是否會疲勞。
